## Föderierter Datenpool

Ein Prototyp für ein föderiertes Verzeichnis sozialer Unterstützungsangebote, das nicht aus einer zentralen Datenbank stammt, sondern in regelmäßigen Abständen durch die Zusammenführung mehrerer dezentraler Datenquellen erstellt wird. Die jeweils verschiedenen Datenformate und Schemata werden durch Mappings zu einem Zielschema, einem semantischen Vokabular, transformiert. Dieses wird im Prozess dynamisch entstehen und wird bestehende Vokabularien in diesen Bereich erweitern.

Das Projekt entsteht im Kontext einer Werkstattbox des [Civic Data Lab](https://civic-data.de/).
